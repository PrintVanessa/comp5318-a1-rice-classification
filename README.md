## Overview

This project implements a complete machine learning pipeline for rice classification, including:

* Data preprocessing (imputation, normalization)
* Multiple classifiers (KNN, Logistic Regression, Naive Bayes, Decision Tree, AdaBoost, Gradient Boosting, Random Forest, SVM)
* Hyperparameter tuning using GridSearchCV
* Model evaluation with stratified 10-fold cross-validation
